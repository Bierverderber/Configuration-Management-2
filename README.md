# Первый этап

Был выполнен первый этап работы. Реализовано получение параметров из файла json и вывод их в консоль. Так же присутствует проверка на соответствие параметров и их наличие (не пустые).

# Второй этап

Был выполнен второй этап работы. Реализована программа для сбора данных о зависимостях пакетов Ubuntu. Программа читает конфигурацию из JSON-файла, загружает индекс пакетов из репозитория и находит указанный пакет. Затем извлекает информацию о прямых зависимостях пакета и выводит их на экран в виде нумерованного списка. Все операции выполняются без использования менеджеров пакетов.
