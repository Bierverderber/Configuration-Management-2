# Первый этап

Реализовано получение параметров из файла json и вывод их в консоль. Так же присутствует проверка на соответствие параметров и их наличие (не пустые).

# Второй этап

Реализована программа для сбора данных о зависимостях пакетов Ubuntu. Программа читает конфигурацию из JSON-файла, загружает индекс пакетов из репозитория и находит указанный пакет. Затем извлекает информацию о прямых зависимостях пакета и выводит их на экран в виде нумерованного списка. Все операции выполняются без использования менеджеров пакетов.

# Третий этап

Добавлен поиск всех зависимостей пакета. Теперь программа находит не только прямые зависимости, но и зависимости зависимостей. Использован простой алгоритм обхода в ширину (BFS) через очередь.
Реализована фильтрация - пакеты с определенной подстрокой в названии игнорируются. Добавлена защита от зацикливания при циклических зависимостях.
Также добавлен тестовый режим для работы с упрощенными файлами. Весь граф зависимостей выводится в понятном виде.

# Четвертый этап

Добавлен поиск обратных зависимостей. Теперь программа показывает не только от каких пакетов зависит указанный пакет, но и какие пакеты зависят от него.
Для этого добавлена функция, которая анализирует все пакеты в репозитории и находит те, у которых указанный пакет есть в списке зависимостей. Результат выводится в том же формате, что и прямые зависимости.
